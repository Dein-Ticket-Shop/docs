title: Überblick
---

Welche Zahlungsmethoden Ihren Kunden zur Verfügung stehen, können Sie sehr einfach im Backend konfigurieren:

1. Klicken Sie oben rechts auf Einstellungen (Zahnrad-Symbol)
2. Wählen Sie im Menü links *Finanzen* aus
3. Klicken Sie bei *Konto- und Finanzdaten* auf *Bearbeiten*
4. Wählen Sie die gewünschten Zahlungsmethoden an oder ab und klicken Sie auf *Speichern*

:::tip Echtzeit-Zahlungsmethoden
Manche Zahlungsmethoden (beispielsweise SEPA Lastschrift) funktionieren nicht in Echtzeit, sondern haben eine Laufzeit von mehreren Tagen. In diesem Fall werden die Tickets erst nach erfolgreicher Zahlung ausgestellt. Damit Kunden auch kurz vor der Veranstaltung noch Tickets kaufen können, muss mindestens eine Echtzeit-Zahlungsmethode aktiviert sein.
:::

---
title: SEPA-Gläubiger ID
---

Damit auf dem Kontoauszug Deiner Kunden der korrekte Name angezeigt wird, müsst Ihr bei unserem Zahlungsdienstleister Stripe eine SEPA-Gläubiger ID hinterlegen.

Falls Ihr keine SEPA Gläubiger ID habt, ist das kein Problem. Die Beantragung ist kostenlos, sowohl möglich für Privatpersonen und Unternehmen und dauert in der Regel nur wenige Minuten. Dazu muss nur ein Online-Formular der Deutschen Bundesbank ausgefüllt werden, danach erhaltet Ihr Eure Gläubiger ID per E-Mail. Hier geht es zur Beantragung der SEPA Gläubiger ID: [https://extranet.bundesbank.de/scp/](https://extranet.bundesbank.de/scp/)

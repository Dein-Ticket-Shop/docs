---
title: Rückerstattung von Barzahlungen
---

Wird eine Veranstaltung abgesagt, so werden dem Saldo der VVK-Stelle die entsprechenden Beträge gutgeschrieben, sodass sie die vollen Ticketkosten den Kunden zurückerstatten können. Die Rückerstattung erfolgt in bar.

Die VVK-Stelle kann folgendermaßen verfolgen, welche Bestellungen sie bereits erstattet hat:

1. Im Veranstalter-Bereich navigiere zu *Bestellungen* und klicke auf *stornierte Bestellungen anzeigen*
2. In der folgenden Liste gibt es eine Spalte *Rückerstattet*. Hier wird ein Datum angezeigt, falls die Bestellung bereits erstattet wurde.

    ![Bestellungstabelle](/img/docs/offices/refunds/cancelled-orders.png)

3. Klicke auf eine Bestellung, um sie als *erstattet* zu markieren, wenn dem Kunden die Rückerstattung ausgezahlt wurde.
4. Bei den Bestelldetails, klicke auf den Button *Betrag zurückerstattet*.

    ![Button](/img/docs/offices/refunds/refund-button.png)

5. Es erscheint eine Bestätigung.
6. In der Liste der Bestellungen wird nun das Datum der Rückerstattung angezeigt, beispielsweise so:

    ![Datum](/img/docs/offices/refunds/refunded-badge.png)

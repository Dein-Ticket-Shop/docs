---
title: Anlegen im Backend
---

1. Klicken Sie oben rechts auf Einstellungen (Zahnrad)
2. Klicken Sie links im Menü auf *Verkaufsstellen*
3. Klicken Sie oben rechts auf *+ Neue Verkaufsstelle*
4. Wählen Sie *Abendkasse* bei *Typ* aus
5. Geben Sie einen Namen ein und eine Adresse. Letztere kann entweder von einem Veranstaltungsort übernommen werden (empfohlen) oder manuell eingegeben werden.
6. Klicken Sie auf Speichern
7. Nun erscheint die Abendkasse in der Liste. Klicken Sie sie an, um weitere Informationen zu sehen.
8. Unter *Aktionen *können Sie das *Anleitungs-PDF generieren*. Es wird empfohlen, dieses auszudrucken und für die Abendkasse bereitzuhalten. Darauf sind die Zugangsdaten, die benötigt werden, um sich anzumelden.

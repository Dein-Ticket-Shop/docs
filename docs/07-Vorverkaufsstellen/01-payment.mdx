---
title: Bezahlung von Umsätzen
---

1. Auf der Übersichtsseite der Veranstalter (https://vvk.dein-ticket.shop) klicken Sie oben rechts auf *Buchhaltung*.
2. Bei der ersten Zahlung an einen Veranstalter müssen Sie zunächst die Zahlungen einrichten. Das erkennen Sie daran, dass in der Tabelle der Veranstalter rechts *zuerst Zahlungen einrichten* steht. Klicken Sie auf *Zahlungen einrichten* oder *Alle Zahlungen einrichten* und folgen Sie den Anweisungen, bis Sie wieder auf der Übersichtsseite der Veranstalter sind.
3. Über die Buttons *jetzt bezahlen* bzw. *Alle bezahlen* können Sie die Zahlungen ausführen. Die Belege werden automatisch erstellt und per E-Mail an Sie versendet.
